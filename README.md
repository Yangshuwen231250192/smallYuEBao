# 小余额宝 (smallYuEBao)

一个模拟余额宝理财功能的轻量级 Web 应用，支持余额转入、转出及每日收益计算（+10%），并提供局域网二维码扫码访问功能。

## 🚀 快速开始

### 1. 安装依赖
克隆项目后，进入目录并安装所需依赖：
```bash
npm install
```


### 2. 启动服务
运行 Node.js 服务器：
```bash
node server.js
```

服务默认启动在 `http://localhost:3000`。

## 📱 手机扫码访问说明
- **局域网限制**：本应用生成的二维码基于**宿主机的局域网 IP**。
- **使用条件**：请确保您的手机与电脑连接在**同一个 Wi-Fi/局域网**内。
- **操作方式**：
    1. 启动服务后，浏览器页面底部会自动生成二维码。
    2. 使用手机浏览器扫描二维码即可访问。
    3. 若自动获取 IP 失败，可在页面底部手动输入电脑的局域网 IP 地址更新二维码。

## 🐳 Docker 部署说明

本项目支持 Docker 容器化部署，但需注意网络环境的差异。

### 构建与运行
```bash
# 1. 构建镜像
docker build -t small-yuebao-image .

# 2. 运行容器 (映射端口 3000)
先获取电脑的局域网 IP
ipconfig
docker run -d -p 3000:3000 -e HOST_IP={YOUR_COMPUTER_IP} small-yuebao-image
```


### ⚠️ 重要注意事项：Docker 环境与扫码访问
由于 Docker 容器运行在独立的虚拟网络环境中，容器内部获取的 IP 地址（通常是 `172.17.x.x`）**无法被局域网内的手机直接访问**。

- **现象**：直接使用上述命令启动后，页面生成的二维码 IP 是容器内部 IP，手机扫码后将无法打开页面。
- **解决方案**：
    - **推荐方式**：直接使用 `node server.js` 在宿主机运行，此时自动获取的 IP 即为局域网可达 IP，扫码功能完美可用。
    - **高级用法**：若必须使用 Docker，需在启动时通过环境变量 `-e HOST_IP=<你的电脑局域网IP>` 强制指定宿主机 IP，并修改代码逻辑以读取该变量。
